The Wide lexer

Wide's lexer produces a Wide range of tokens, which are tokenized from an input range of characters with an initial position. The input range should not skip any whitespace as this can have lexical meaning. The lexer interface is defined in Wide/Lexer/Lexer.h. Formally, the Wide lexer only operates on ASCII, but comments and strings may contain arbitrary data.
