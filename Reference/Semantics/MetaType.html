Meta types

A Meta type is a classification of types in the Wide type system. Meta types are primitive types. All meta types are also default-constructible constants.

Currently, constructor types, C++ namespaces, templates, modules, decltype(null) and decltype(cpp) are all meta types. Some implementation details are also meta types.
